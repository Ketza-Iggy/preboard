How QR codes work with this site: Each QR code is just a link to this page with the room number pre-filled. You generate one QR per room, print and laminate it, and tape it to the door frame. The foreman scans it and the room number auto-loads into the form.
Step-by-step setup
1
Copy this site's URL from your browser address bar after uploading to GitHub Pages. It will look like:
https://yourname.github.io/preboard/
2
Add room parameters to the end of the URL for each room. For example, Room 312 on Level 3:
https://yourname.github.io/preboard/?floor=Level+3&room=312&type=Residential+unit
The form will auto-fill the floor, room number, and room type when opened.
3
Generate QR codes for free at qr.io or qr-code-generator.com. Paste each room URL in, download the QR image.
4
Print and laminate. Include the room number in large text on the card. Tape to the door frame before framing inspection begins.
5
Share the dashboard URL with your GC and PM so they can monitor room status and respond to deficiencies from any device.

Uploading to GitHub Pages (free)
1
Go to github.com and create a free account if you don't have one.
2
Click New repository. Name it preboard. Make it Public. Click Create.
3
Click uploading an existing file, drag the downloaded HTML file in, name it index.html, and click Commit.
4
Go to Settings → Pages. Under Source, select main branch. Click Save. Your site will be live at yourname.github.io/preboard within 2 minutes.
